Q: IL Programma è sicuro?

A: Si il programma è open-source ed è stato scritto in C# l'exe è stato creato tramite il compilatore di Visual Studio

Q:Quando modifico un settaggio posso tornare indietro?

A:Si il programma permette sempre il ripristino alle impostazioni di default

Q:Le iso sono sicure?

A:Si le iso contengono il codice SHA256, e per le iso modificate, le relative modifiche e script sono tutti opensource
